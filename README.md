# source-code-compiler
This is an app for compiling your code with various programming language. The project is built with ReactJS, CSS and HTML.

# How to run the project
1. Clone the repository
2. Go to your repository directory in you CLI
3. Run `npm install` in CLI for install all package needed
4. Run `npm start` for running the project
5. If the program is run, you can go to the browser using address `localhost:3000`
